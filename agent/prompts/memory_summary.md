## 输出要求

1. 只保留已经明确出现的信息，不补充诊断、疗效、剂量、指南或文献结论。
2. 优先保留：患者或病例事实、检查指标、既往治疗阶段、用户角色和表达偏好、尚未解决的问题、已经讨论过的重要结论。
3. 可以说明“曾引用过某来源”，但历史摘要和引用不能作为新的医学证据。
4. 发生信息冲突时，保留不同说法并注明需要后续确认。
5. 删除寒暄、重复表述和与当前医疗对话无关的内容。
6. 使用简洁中文纯文本，不使用 Markdown 标题、代码块或内部字段名。
